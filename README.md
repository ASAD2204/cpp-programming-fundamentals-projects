## üöÄ Getting Started

### Prerequisites
- C++ Compiler (GCC, MinGW, Visual Studio, or Code::Blocks)
- Basic understanding of C++ programming
- Windows/Linux/Mac OS

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/ASAD2204/cpp-programming-fundamentals-projects.git
cd cpp-programming-fundamentals-projects
```

2. **Navigate to any project folder**
```bash
cd "Student Management System"
```

3. **Compile the program**
```bash
g++ main.cpp -o program.exe
```

4. **Run the executable**
```bash
./program.exe
```

## üìö Project Details

### 1. Number Guessing Game
Simple interactive game where the computer generates a random number and the player has to guess it.
- **Concepts**: Random number generation, loops, conditional statements
- **Difficulty**: Beginner

### 2. Simple Calculator
Perform basic arithmetic operations including addition, subtraction, multiplication, and division.
- **Concepts**: Switch statements, functions, error handling
- **Difficulty**: Beginner

### 3. Student Grade Calculator
Calculate student grades based on marks and determine letter grades and GPA.
- **Concepts**: Arrays, functions, grade calculation logic
- **Difficulty**: Beginner

### 4. Tic-Tac-Toe Game
Classic two-player game with win detection and draw handling.
- **Concepts**: 2D arrays, game logic, win condition checking
- **Difficulty**: Beginner

### 5. Unit Converter
Convert between various units (length, weight, temperature, etc.).
- **Concepts**: Functions, mathematical operations, menu systems
- **Difficulty**: Beginner

### 6. Student Management System
Complete student record management with file persistence.
- **Features**: Add/View/Update/Delete students, GPA calculation, sorting
- **Concepts**: Structs, file I/O, array manipulation, sorting algorithms
- **Difficulty**: Intermediate

### 7. Library Book Management
Comprehensive library system with book tracking and issue/return functionality.
- **Features**: Book inventory, issue/return books, category search, availability tracking
- **Concepts**: Multiple structs, date handling, complex file operations
- **Difficulty**: Intermediate

### 8. Bank Account Manager
Banking application with account management and transaction tracking.
- **Features**: Account creation, deposits, withdrawals, transfers, PIN security, transaction history
- **Concepts**: Multi-file handling, security implementation, transaction logging
- **Difficulty**: Advanced

### 9. Expense Tracker
Personal finance management application with detailed analytics.
- **Features**: Category-wise expenses, monthly reports, profit analysis, export functionality
- **Concepts**: Data analysis, report generation, file export
- **Difficulty**: Intermediate

### 10. Contact Book
Contact management system with advanced search capabilities.
- **Features**: Add/Edit/Delete contacts, favorites, category filtering, export contacts
- **Concepts**: String manipulation, search algorithms, data categorization
- **Difficulty**: Intermediate

### 11. Inventory Management System
Professional inventory system with sales tracking and profit analysis.
- **Features**: Product CRUD, stock management, low stock alerts, sales history, profit calculations
- **Concepts**: Multiple file handling, business logic, reporting, data analytics
- **Difficulty**: Advanced
